# Hotel Advantage Inn Prayagraj - Official Website

Welcome to the official, highly optimized website codebase for **Hotel Advantage Inn, Civil Lines, Prayagraj**. 

This website has been built and engineered with modern web standards, focusing on high speed, premium visual aesthetics, responsive mobile design, and advanced search engine readiness (SEO, AEO, GEO, and CRO).

---

## 🚀 Features

* **🏎️ Ultra-Fast Load Speeds**: Optimized using next-gen `.webp` images, explicit width/height dimensions (Zero CLS), and deferred font/icon loading.
* **📱 100% Mobile Friendly**: Responsive layouts, touch-optimized elements, sliding mobile drawer menu, and a sticky mobile action CTA bar (Call/WhatsApp).
* **🤖 Advanced AEO/GEO Schema**: 86 verified structured schema checks passed. Embedded JSON-LD schemas representing `Organization`, `Hotel`, `LocalBusiness`, `HotelRoom`, `Restaurant`, and `TouristAttraction` for search engines like Google, Bing, and AI search tools.
* **✨ PWA Ready**: Configured with `manifest.json` and a service worker (`sw.js`) for offline capabilities and mobile home screen installation.
* **🛡️ Hardened Security**: Implementation of Content Security Policy (CSP), Referrer-Policy, and tab-nabbing protections (`rel="noopener noreferrer"`).

## 🌐 How to Deploy on GitHub Pages

This codebase is 100% ready for **GitHub Pages** out of the box because it uses relative paths. Follow these simple steps to go live:

1. **Upload Files**: Create a new GitHub repository (public or private) and upload all files from this directory to the `main` branch.
2. **Enable Pages**:
   * Go to your repository settings on GitHub.
   * On the left sidebar, click on **Pages**.
   * Under **Build and deployment**, select **Deploy from a branch**.
   * Under **Branch**, select `main` (or `master`) and `/ (root)`. Click **Save**.
3. **Go Live**: After a few seconds, GitHub will provide you with your live URL (e.g., `https://username.github.io/repository-name/`).
